# OpenToad 功能清单

## 核心功能

| 功能 | 说明 |
|------|------|
| 设置 (Settings) | 配置 API Key、Provider、模型等 |
| 清空 (Clear) | 清空对话历史 |
| 关于 (About) | 显示版本信息 |
| 对话框 (Dialog) | 设置弹窗 |

## 聊天功能

| 功能 | 说明 |
|------|------|
| 消息发送 | 输入框 + 发送按钮 |
| 消息显示 | 区分用户/AI/系统消息 |
| 流式输出 | 打字机效果 |
| 错误提示 | 错误消息显示 |

## 系统集成

| 功能 | 说明 |
|------|------|
| 状态栏 | 显示连接状态 |
| 工具栏 | 快捷操作入口 |
| 系统托盘 | 后台运行 |

## UI 交互

| 功能 | 说明 |
|------|------|
| 窗口最小化/最大化 | 基础窗口控制 |
| 消息滚动 | 自动滚动到底部 |
| 输入回车发送 | 快捷键支持 |
| 加载状态 | 发送时禁用按钮 |
