## 三、核心特性

### 1. 多模型支持

- **海外模型**: Claude, GPT, Gemini
- **国内模型**: 文心一言, 通义千问, 混元, ChatGLM, Kimi, DeepSeek
- **自部署模型**: Ollama, LocalAI, LM Studio, vLLM

### 2. Tool 系统

| Tool | 说明 |
|------|------|
| Shell | 执行 shell 命令 |
| Filesystem | 文件读写操作 |
| Web Search | 网页搜索 |
| Calculator | 数学计算 |

### 3. ReAct 推理

- 思维链可视化
- Tool 调用编排
- 迭代执行机制

---

## 四、部署形态

### 1. 客户端分发

- Windows / macOS / Linux 桌面应用
- 一键安装，自动注册实例

### 2. 自部署服务

- 用户自行搭建服务端
- 支持私有化部署

### 3. 云端服务

- 官方托管服务
- 实例自动接入
- 任务分发赚取 Token
