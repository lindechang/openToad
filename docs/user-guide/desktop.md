# OpenToad 桌面端使用说明

## 📖 简介

OpenToad 桌面端是一个跨平台的图形界面应用，基于 PySide6 开发，支持 Windows、macOS 和 Linux。提供完整的 AI 助手功能，包括对话、记忆管理、Agent 协作和手机连接。

## 🚀 快速开始

### 安装依赖

```bash
# 安装核心依赖
pip install -e .

# 安装桌面应用依赖
pip install -e apps/desktop
```

### 启动桌面应用

```bash
# 方式 1: 使用启动脚本
./opentoad.sh  # macOS/Linux
opentoad.bat   # Windows

# 方式 2: 直接运行
python3 apps/desktop/src/main.py  # macOS/Linux
python apps/desktop/src/main.py   # Windows
```

## 💬 基本功能

### 开始对话

1. 在输入框中输入你的消息
2. 按 Enter 键发送
3. AI 会实时流式回复
4. Shift+Enter 可换行输入

### 管理会话

| 操作 | 说明 |
|------|------|
| 新建会话 | 点击 "新建会话" 按钮 |
| 切换会话 | 在会话列表中点击会话 |
| 删除会话 | 右键会话，选择删除 |
| 重命名会话 | 右键会话，选择重命名 |

### 切换记忆体

1. 点击会话列表顶部的记忆体下拉菜单
2. 选择要使用的记忆体
3. AI 将使用该记忆体中的身份和记忆

## ⚙️ 设置

### LLM 配置

#### 选择提供商

打开设置 → LLM 配置，选择 LLM 提供商：

| 提供商 | 说明 |
|--------|------|
| Anthropic | Claude 系列模型 |
| OpenAI | GPT 系列模型 |
| DeepSeek | DeepSeek 模型 |
| 通义千问 | 阿里巴巴模型 |
| 文心一言 | 百度模型 |
| 混元 | 腾讯模型 |
| ChatGLM | 智谱模型 |
| Kimi | 月之暗面模型 |
| Gemini | Google 模型 |
| Ollama | 本地模型 |

#### 配置 API Key

1. 选择提供商后
2. 在 API Key 输入框中输入你的 Key
3. 点击保存
4. API Key 会被加密存储在本地

#### 选择模型

每个提供商支持多个模型：

**Anthropic 模型**：
- claude-sonnet-4-20260220 (推荐)
- claude-opus-4-20260220
- claude-3-5-sonnet-20241022

**OpenAI 模型**：
- gpt-5.4 (推荐)
- gpt-5.4-pro
- gpt-4o

**DeepSeek 模型**：
- deepseek-chat (推荐)
- deepseek-reasoner
- deepseek-coder

### 流式输出

- **启用** - 实时显示 AI 回复（打字机效果）
- **禁用** - 等 AI 回复完后一次性显示

### 温度配置

调节 AI 的创造性：

| 温度值 | 效果 |
|--------|------|
| 0.1-0.3 | 更确定、更准确 |
| 0.5-0.7 | 平衡（推荐） |
| 0.8-1.0 | 更有创意 |

## 🤖 Agent Network

### 打开 Agent 面板

点击左侧图标栏的 "🤖" 按钮，打开 Agent Network 面板。

### Agent 列表

显示所有在线 Agent：

| 角色 | 图标 | 职责 |
|------|------|------|
| Coordinator | 🎯 | 协调任务分配和结果汇总 |
| Worker | ⚙️ | 执行具体任务 |
| Researcher | 🔍 | 进行研究和分析 |
| Writer | ✍️ | 撰写文档和内容 |
| Reviewer | 👁️ | 审查和评估结果 |

### 任务管理

- **创建任务** - 点击创建新任务按钮
- **查看任务** - 在任务列表中查看所有任务
- **任务状态** - 待处理/进行中/已完成
- **进度条** - 可视化任务完成度
- **分配任务** - 将任务分配给特定 Agent

### 共享工作区

- **创建工作区** - 新建协作工作区
- **添加笔记** - 在工作区中添加协作笔记
- **文件共享** - 上传和共享文件
- **活动时间线** - 查看所有协作活动
- **参与者管理** - 添加/移除工作区参与者

## 🔐 账号系统

### 注册账号

1. 点击左侧图标栏的 "🔐" 按钮
2. 点击 "注册"
3. 填写邮箱、密码和昵称
4. 点击确认

### 登录账号

1. 打开账号面板
2. 输入邮箱和密码
3. 点击登录

### 账号功能

登录后可以：
- 绑定记忆体到账号
- 加密保护记忆体
- (未来) 云端同步记忆体

## 📱 Gateway - 手机连接

### 启用 Gateway

1. 打开设置 → 📱 手机连接
2. 勾选 "启用 Gateway 服务"
3. 选择端口号（默认 18989）

### 手机连接方式

手机 App 可以通过以下地址连接：

```
ws://电脑IP:18989/ws
```

### 认证

连接时需要提供 Instance ID 进行认证。

### Gateway 功能

- 手机直接访问 AI 助手
- 无需经过云端服务器
- 支持流式输出
- 共享相同的记忆系统

- 删除不需要的记忆

## 🎯 高级功能

### 多 Agent 协作

1. 打开 Agent Network 面板
2. 创建一个任务
3. 任务会自动分配给合适的 Agent
4. 多个 Agent 协同工作
5. 结果汇总并返回给你

### 工作区协作

1. 创建共享工作区
2. 添加参与者
3. 在工作区中添加笔记和文件
4. 所有参与者都可以查看和编辑
5. 完整的活动记录

### 身份与凭证

- **Agent 身份** - 每个 Agent 有唯一身份
- **信誉评分** - 根据任务完成情况评分
- **凭证系统** - 颁发能力证书
- **信任等级** - 根据信誉自动计算

## 🚀 提示

### 第一次使用

1. 启动桌面应用
2. 打开设置 → LLM 配置
3. 选择你的 LLM 提供商
4. 输入 API Key 并保存
5. 设置 AI 名字和角色
6. 开始对话！

### 记忆管理技巧

- 使用 "/remember" 在对话中记住重要信息
- 重要的知识会自动升级为长期记忆
- 可以在设置中手动管理所有记忆

### Agent 协作

- 复杂任务可以分解给多个 Agent
- Researcher 负责研究，Writer 负责撰写
- Coordinator 会自动协调整个过程

### 安全建议

- API Key 会被加密存储，不要分享
- 登录后记忆体会被加密保护
- 可以随时登出和切换账号

## ❓ 常见问题

### 找不到设置在哪里？

点击左侧图标栏的 "⚙️" 按钮。

### 如何切换 LLM 模型？

在设置 → LLM 配置中选择。

### 记忆体在哪里存储？

默认在 `~/.opentoad/memory.db`，可以在设置中查看和更改。

### 如何重置应用？

删除 `~/.opentoad` 文件夹即可（注意备份记忆体！）。

### 支持的平台？

- Windows 10/11
- macOS 10.14+
- Linux (Ubuntu 18.04+)
